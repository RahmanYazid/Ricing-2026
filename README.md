# Ricing-2026

Dotfiles for an Arch Linux + Hyprland setup featuring a dynamic theme based on pywal.

---

````bash
## Dependencies

\```
yay -S hyprland waybar rofi dunst kitty fish python-pywal awww neovim pokemon-colorscripts-git eza starship
\```

````

---

## Installation

Clone the repository:

```bash
git clone git@github.com:RahmanYazid/Ricing-2026.git ~/dotfiles

```

Copy the configurations to `~/.config`:

```bash
cp -r ~/dotfiles/hypr ~/.config/
cp -r ~/dotfiles/waybar ~/.config/
cp -r ~/dotfiles/rofi ~/.config/
cp -r ~/dotfiles/dunst ~/.config/
cp -r ~/dotfiles/kitty ~/.config/
cp -r ~/dotfiles/fish ~/.config/
cp -r ~/dotfiles/nvim ~/.config/

```

Make the script executable:

```bash
chmod +x ~/.config/hypr/scripts/theme-switch.sh

```

Set up pywal with your wallpaper of choice:

```bash
wal -i /path/to/wallpaper.jpg

```

---

## Theme Switching

The `theme-switch.sh` script changes the wallpaper while simultaneously updating the color scheme via pywal and awww-daemon. It can be run manually or triggered via a keybinding in `hyprland.conf`.

## Notes

- Neovim uses LazyVim.
- All colors are automatically generated by pywal from the active wallpaper.
- A wallpaper picker is available via rofi.
- Set fish as your default shell: `chsh -s /usr/bin/fish`
